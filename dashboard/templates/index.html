About the Scraper

Polls LinkedIn + Indeed for jobs posted in the last 2 hours. Deduplicates across sites, scores by relevance, and appends only new jobs to the CSV. A Windows Task Scheduler job also runs this every hour automatically.

Quick Reference

Scheduler setup
Run once in an elevated PowerShell: 

schtasks /Create /TN "JobSpyHourly" `
  /TR "powershell -NonInteractive -ExecutionPolicy Bypass -File E:\JOBS\run_hourly.ps1" `
  /SC HOURLY /MO 1 /ST 00:00 /RL HIGHEST /F

Setup resume profile

python resume_builder/tailor.py setup

Edit keywords directly in run_jobspy.py → PROFILE dict.
